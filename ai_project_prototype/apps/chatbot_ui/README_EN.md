# Streamlit & FastAPI Connection Test

A simple project demonstrating communication between a **Streamlit** frontend and a **FastAPI** backend.

---



# FastAPI

Provides the backend service and exposes HTTP APIs for the Streamlit frontend.

Designed for high performance with asynchronous request handling.

```text
Receive Request
        ↓
Process Business Logic
        ↓
Invoke LLM
        ↓
Return Response
```
