# Streamlit & FastAPI Connection Test

A simple project demonstrating communication between a **Streamlit** frontend and a **FastAPI** backend.

---



## FastAPI

FastAPI serves as the backend service, exposing HTTP APIs for communication with the frontend.

It is designed for high performance and supports asynchronous request handling.

```text
Receive Request
        ↓
Process Business Logic
        ↓
Invoke LLM
        ↓
Return Response
```

---



## Streamlit

Streamlit is a rapid application framework for building interactive web interfaces with Python.
